# 大富翁

基于经典大富翁规则的本机 2–4 人对战游戏。3D 棋盘与物理骰子，地产格为中国城市名；开局可选棋盘大小，城市随机生成。

## 功能

- **五档棋盘**：迷你 24 / 小号 32 / 中号 40 / 大号 48 / 超大 56 格（默认中号）
- **中国城市地产**：从城市池随机抽取；**北京、上海、广州、深圳**每局常驻且价格最高
- **完整地产循环**：起点、城市、铁路、公用事业、税收、机会 / 社区基金、监狱、免费停车
- **回合制对战**：掷骰、移动、买地 / 拍卖、收租、建房、抵押、交易、破产判定
- **3D 场景**：React Three Fiber 棋盘、动物棋子、Rapier 物理骰子
- **可调规则**：大厅可改开局金币（默认 1500）与过起点金币（默认 200）
- **本机热座**：同一浏览器内轮流操作，无需联网

### 与标准规则的差异

- 城市地产**无色组**：不要求成套即可建房，空地也不因垄断而双倍租金
- 建房仅在**落地该格**时有一次机会（停靠格建房）
- 棋子为动物形象：小猪 / 猫咪 / 兔子 / 小熊

## 快速开始

需已安装 [Node.js](https://nodejs.org/) 与 [pnpm](https://pnpm.io/)。

```bash
pnpm install
pnpm dev
```

浏览器打开终端提示的本地地址即可开局。若需指定 npm 源：

```bash
pnpm install --registry https://registry.npmjs.org/
```

## 脚本

## 目录结构

```
src/
├── game/           # 纯逻辑：棋盘、卡牌、规则引擎、随机数
│   ├── board.ts    # 棋盘生成、规格与常量
│   ├── cities.ts   # 城市名池与北上广深常驻
│   ├── cards.ts    # 机会 / 社区基金
│   ├── engine.ts   # reduce(state, action)
│   ├── types.ts    # 类型与阶段机
│   └── rules/      # 租金、建房等辅助
├── store/          # Zustand：游戏态、相机焦点
├── scene/          # R3F：棋盘、棋子、建筑、骰子
├── ui/             # 大厅、HUD、各类弹窗
├── App.tsx
└── main.tsx
```

规则与渲染分离：`src/game` 无 React / Three 依赖，便于单测与后续扩展。

## 游玩流程

1. 大厅选择人数（2–4）、**棋盘大小**、棋子形象、开局 / 过起点金币，开始游戏
2. 本局按所选规格生成棋盘，城市地产随机抽取（北上广深固定为最高价四格）
3. 当前玩家掷骰 → 棋子移动 → 落地结算（购买、租金、抽卡、入狱等）
4. 可在行动栏管理资产、发起交易；拒绝购买时进入拍卖
5. 其余玩家破产后判定胜者，或点「结束游戏」返回大厅
